Project 3

Adding in a custom home screen using XML, also known as working with a SVG file. For this, you will be utilizing the BlackBerry Composer. This is a quick and effective way to get your XML coded, but, you will still need to know some XML as you will have to hand code in the specifics. This custom home screen will be made for the BlackBerry Bold 9700 and will feature an animated hidden dock of ten mini icons, opposed to the traditional six static icons.

Once composer is open, start placing in default icons (this will allow the device to use device ordered icons). You all of those are in place, you will need to replace all the images from full size "80 x 80" icons with half sized icons; "40 x 40" (keep the default icon image), this includes the icon focus image. Now, with all of the icon images changed, align (and keep them in order 1-10) icon one to the far left bottom corner. Take icon 10 and place it in the far right bottom corner. Highlight all your icons and use the align tool to align them all to the bottom, then space them using the space align tool. That's all you need to do in BlackBerry Composer. Export and save your SVG file so you can access it easily.

Now, locate the SVG file you saved and open it up in a text editor of your choice. You will need to input some code. Luckily, you can do this at the very bottom of the XML (SVG) file. Scroll all the way down and make some space right before your closing tag. You will need to add in this exact code to perform the animated hidden dock interaction:

<animateTransform xlink:href="#Layer_1" id="_anim_l1" attributeName="transform" type="translate"
to="0,0" dur="0.5s"
fill="freeze"
begin="accessKey(ESCAPE)" />
<animateTransform xlink:href="#Layer_1" id="_anim_l2" attributeName="transform" type="translate"
to="0,-80" dur="0.5s"
fill="freeze"
begin="accessKey(SPACE)" />

Saved? Good. Import this SVG file into the BlackBerry Theme Studio, set the icon navigation to horizontal. Replace wallpapers if desired (home screen can be done on device). Now, export this theme to your device and enjoy! Can this SVG file be used for other devices?

  • Compatible Devices
  • BlackBerry Tour 9630
  • BlackBerry Bold 9650
  • BlackBerry Curve 8900
  • Not Compatible Devices
  • BlackBerry Curve 8300
  • BlackBerry Curve 9300
  • BlackBerry Storm 9350

An animated hidden dock allows the user to see the icon dock slide down and off the screen before it hides. Hiding the icons means that nothing is selectable while they are hidden. SVG stands for Scalable Vector Graphics. XML stands for Extensible Markup Language.
